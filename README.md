# Snake Game (Jogo da Cobrinha)

Este projeto é uma implementação do clássico Jogo da Cobrinha (Snake) utilizando HTML, CSS e JavaScript puro.

## Como jogar

- Use as setas do teclado (↑, ↓, ←, →) para controlar a direção da cobrinha.
- O objetivo é comer o máximo de alimentos possível, aumentando o tamanho da cobra e sua pontuação.
- O jogo termina se a cobra colidir com as paredes ou consigo mesma.
- Após o game over, clique em "jogar novamente" para reiniciar.

## Funcionalidades

- Interface moderna e responsiva.
- Pontuação dinâmica.
- Alimento com cor aleatória a cada rodada.
- Tela de game over com pontuação final.

## Tecnologias utilizadas

- HTML5
- CSS3 (com fonte retrô)
- JavaScript (lógica do jogo, renderização e controles)

## Como executar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Divirta-se!

## Estrutura do projeto

- `index.html`: página principal do jogo.
- `css/styles.css`: estilos visuais.
- `js/script.js`: lógica e funcionamento do jogo.
